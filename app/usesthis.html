The Setup

This is what I use to get stuff done

Hardware

  • Ikea Markus
    High-end desk chair from Ikea.

  • MacBook Pro
    (Retina, 13-inch, Mid 2014), work computer. This is where most of the software development gets done. I usually have it attached to an external Dell display at the CartoDB office.

  • MacBook Air
    (11-inch, Late 2010), home computer. I bought it with my first payroll. Best computer I ever had.

  • Dell S2309W
    When it comes to displays I always trust Dell. This was the lower-end screen I could afford at the time and has accompanied me since then.

  • Logitech Anywhere MX
    I'm used to touchpad gestures, but I easily jump to the mouse for some tasks. This is the only wireless mouse I've come to like, after a couple of years. I just can't Magic Mouse. — Amazon

  • Audio-Technica ATH-M50
    Red goes faster. After some research these were some of the best over-ear headphones, and they are worth every dollar. — Amazon

  • Moto E
    (2nd Gen.), I've learnt to know I don't really need the latest iPhone. Despite I really miss a better camera (it doesn't even have Flash), the rest is good enough so I would't mind breaking or losing it. — Amazon

  • Garmin Fenix
    I'm not riding the smartwatch wave yet, but I like to track my runs and rides. It provides the classic ABC, plus GPS. One day I'll wear a Speedmaster Professional Moonwatch.

After using one during my last holiday trip, I'm planning to buy a Kindle Paperwhite.

Software

Beware, entering the Nerd zone.

  • Sketch
    Haven't paid for a personal license yet. I'm still broken-hearted after supporting version 2, but we fully use it at CartoDB for design.

  • Sublime Text
    Coming from the now Open Source software TextMate, Sublime Text is the fastest and simplest code editor I've used. From time to time I give Atom a try but I still don't feel comfortable. I use it with Spacegray theme, and Source Code Pro font. The only vim command I know is save and exit.

  • iTerm 2
    Along with ohmyzsh. Chruby for Ruby (and Rubygems) version management. I use Grunt for my Front-end tooling workflow, and Jekyll for building static sites (like this). Virtual Machines live in vagrant.

  • Google Chrome
    The usual stuff here, I spend half of the day in DevTools. I try to keep it low on extensions: Blipshot for screenshots (Awesome screenshot was better but I don't like their malicious practices), Earth View for new tabs (my favourite extension ever), Google Cast as a companion for the Android Chromecast app, Instapaper for saving web pages to read later, JSONView (there are savages out there who read plain JSON responses), LastPass for password management, Quote.Plus to tweet quotes with one click, and uBlock to prevent tracking.

  • Other software and services
    There's a bunch of free and subscription-based software I use on a daily basis: ImageOptim, Spectacle, GitHub, Gmail, Slack, Wunderlist, Simplenote, Digg Reader, Pinboard, Strava, Spotify, Netflix, Plex, et cetera.

Colophon

This website has been built using the software and hardware above: it's based on Jekyll and Grunt, and hosted on AWS S3 with Route 53. You can find the source code in GitHub.
